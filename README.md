# Quick Start

### Add a .env file 

```
MONGO_URL= 
CLOUDINARY_CLOUDNAME = 
CLOUDINARY_APIKEY = 
CLOUDINARY_APISECRET = 
SECRET_KEY= 
PORT = 5000
```

### Install server dependencies

```
npm install
```

### Install client dependencies

```
cd client
npm install
```

### Run both Express & React from root

```
npm run dev
```

### Debug server

```
npm run debug
```

### Build for production

```
cd client
npm run build
```
